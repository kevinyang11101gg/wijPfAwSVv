# Python房价预测系统

## 核心功能
- 登录注册
- 房屋数据可视化大屏
- 房产信息展示
- 房屋信息管理
- 房屋资讯浏览
- 留言反馈
- 用户管理
- 个人中心
- 系统管理

## 适用场景
- 房地产市场分析
- 房价趋势预测
- 房产投资决策
- 房地产企业内部管理

## 技术特点
- 基于Django框架开发
- 使用MySQL数据库存储数据
- 前端采用Vue技术实现交互
- 系统稳定可靠，易于部署和使用
